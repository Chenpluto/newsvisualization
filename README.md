#### 什么是反向代理
反向代理（Reverse Proxy）是一种代理服务器的形式，在计算机网络中起到中间层的作用。它接受客户端的请求，然后将请求转发给内部网络上的服务器，
并将从服务器上得到的结果返回给客户端。与传统的正向代理（Forward Proxy）不同，反向代理==隐藏了真实的服务器信息==，客户端只知道与反向代理通信，
而==不知道真实服务器的存在==。
